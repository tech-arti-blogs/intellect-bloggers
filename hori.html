Horizontal Scalability is a provision in which along with the existing resources multiple resources either hardware or software can be added which can be seen as a single logical unit. Some of the resources such as servers can be connected. Example: While hosting a web page, if there is no traffic as there are many users adding more servers will increase the speed. But all these servers will perform the same task. This process is also called as Scaling out.
