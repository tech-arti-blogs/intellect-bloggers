Various Cyber Security techniques are:

1) Password Security and Access control

The most common way to protect our data is to control the anonymous access by providing username and password. Strong passwords are recommended to provide security. Strong passwords are the mix of numbers, special characters and letters.

2) Authentication of data

3) Malware Scanners

Malware generally refers to worms, Trojan horses and other harmful viruses. Malware scanners are software that scans the files in our systems for malware.

4) Firewall

A firewall is a hardware or software that prevents hackers, worms, viruses to reach the computer. Firewall acts a wall where all messages sent or received over internet are verified and those messages with security issues are blocked.

5) Anti-virus reference

Anti-virus software is a code that removes malicious code from the system. Most anti-virus have auto-update feature which displays the details of new viruses automatically.
