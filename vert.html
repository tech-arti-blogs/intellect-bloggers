Vertical scalability is a provision in which along with existing hardware more resources can be added. In the same server/ network we can add more resources. Example: For performing any task, if the processing power is less, then any hardware, servers can be added to it to increase its processing power. To perform a single task multiple resources is added. This form of scalability is called as Scaling up or Scaling in.
